The Yoda_powers module
======================

:author: Sebastien Ravel
:contact: sebastien.ravel@cirad.fr
:date: 22-07-2020
:version: 1.0.0

Use it to import very handy functions.

Example:

>>> from yoda_powers import dict2txt
>>> dico = {"key1":"value1","key2":"value2","key3":"value3"}
>>> dict2txt(dico)
key1	value1
key2	value2
key3	value3

Required Module install
-----------------------

This module run with Python 3.x and not Python 2.x

** Include module of python:**
    - argparse,
    - os,
    - subprocess,
    - glob,
    - re,
    - warning,
    - pathlib,
    - collections,
    - gzip,
    - urllib,

** Module need to install before use:**
    - BioPython
    - pyvcf
    - pyfaidx
